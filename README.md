# Desenvolvimento Web

Repositorio de estudos e praticas em desenvolvimento web, criado durante minha formacao inicial na area de tecnologia.

## Objetivo

Registrar minha evolucao nos fundamentos de desenvolvimento front-end, praticando estrutura de paginas, estilizacao, responsividade e primeiros recursos com JavaScript.

## Conteudos estudados

- HTML5
- CSS3
- JavaScript
- Estrutura de paginas web
- Organizacao de arquivos
- Responsividade
- Git e GitHub

## Importancia para minha formacao

Este repositorio mostra minha base de aprendizado em desenvolvimento web e serve como registro dos exercicios feitos durante o curso. Ele faz parte da minha preparacao para oportunidades de estagio ou vaga junior em tecnologia.
